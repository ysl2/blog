通过pip可以安装

```bash
# Install
pip install pyqt5-tools

# Use
# Method 1: directly
designer
# Method 2: command
pyqt5-tools designer
```


下面的方法是系统级别，不需要这么麻烦。

```
sudo apt-get install qttools5-dev-tools qttools5-dev
```
