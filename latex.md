# latex

## Installation

### Linux

```bash
sudo apt install texlive-full
```

### Windows

1. Download: <https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/Images/texlive.iso>

2. Install

3. Add to PATH: `C:\texlive\2023\bin\windows`
